# svelte-oauth-example
##### It is a simple oauth system made with Pocketbase. Follow the steps below for installation.
## Setup (Project)

1) Download source codes.
```bash
https://github.com/Queaxtra/svelte-oauth-example
```

2) Install.

```bash
  npm install
```

3) Run the site.

```bash
  npm run dev
```

## Setup (Pocketbase)
